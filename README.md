A modern World Cup betting pool platform inspired by sportsbook UIs.

Users can:

Create and manage predictions
Track rankings in real time
View match schedules and results
Compare predictions with friends
Pay access fees using PIX + QR Code via Mercado Pago

Built as a full-stack portfolio project focused on modern web architecture, APIs, authentication, payments, CI/CD, and scalable application structure.
